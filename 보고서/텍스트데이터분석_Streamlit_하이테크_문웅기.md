# 📝 [보고서] Streamlit(스트림릿)

## 1. Streamlit(스트림릿)이란 무엇인가요? 🚀
Streamlit은 **데이터 과학자나 인공지능(AI/ML) 엔지니어들이 단 몇 줄의 파이썬 코드만으로 역동적인 데이터 앱(App)을 만들 수 있게 해주는 오픈 소스 프레임워크** 
복잡한 웹사이트 제작 기술을 몰라도, 단 몇 분 만에 아주 강력하고 훌륭한 데이터 앱을 뚝딱 만들어서 인터넷에 배포할 수 있다

## 2. Streamlit 하기 위한 3단계 핵심 과정 🗺️

| 단계 | 과정 이름 | 설명 |
| :--- | :--- | :--- |
| **1단계** | **시작하기 (Get started)** | 내 컴퓨터에 Streamlit을 **설치(Installation)**하고, 어떻게 작동하는지 **기본 개념(Fundamentals)**을 배운다|
| **2단계** | **개발하기 (Develop)** | **API 레퍼런스**를 보며 각 기능이 어떻게 쓰이는지 배운다 |
| **3단계** | **배포하기 (Deploy)** |  **Streamlit Community Cloud**나, 기업용 통합 시스템인 **Snowflake**를 이용해 앱을 이용 |

## 3. 내 앱을 예쁘고 신기하게 꾸며주는 최신 기능들 ✨
최근 Streamlit에는 앱을 훨씬 더 다채롭게 만들어주는 마법 같은 기능들이 많이 추가되었다.

| 기능명 | 하는 역할 |
| :--- | :--- |
| **동적 컨테이너** (`st.tabs`, `st.expander`) | 화면을 여러 개의 탭으로 깔끔하게 나누거나, 클릭하면 숨겨진 내용이 스르륵 펼쳐지는 패널을 만들 수 있다. |
| **클릭 가능한 이미지** (`st.image`) | 이미지에 링크를 달아서, **사진을 클릭하면 다른 웹사이트로 이동**(HTTP/HTTPS URL)하게 만들 수 있다. |
| **알록달록 글씨 색상** (Markdown) | 글을 쓰는 마크다운 기능에서, 텍스트의 글자색과 배경색을 **원하는 디자인(CSS 색상)으로 마음껏 예쁘게 꾸밀 수 있다.** |
| **숨겨진 페이지** (`st.Page`) | 화면 옆의 메뉴(내비게이션)에서는 보이지 않지만, 나만 아는 주소로 접속할 수 있도록 **페이지를 몰래 숨겨주는** 기능이다. |


# 🛠️ 화면을 깔끔하게 정리하는 `st.tabs`와 `st.expander`

## 1. 동적 컨테이너(Dynamic containers)란? ✨
`st.tabs`와 `st.expander`는 화면을 여러 구역으로 나누어 역동적이고 깔끔하게 만들어주는 **동적 컨테이너**이다.

## 2. 최신 마법 도구: `on_change` 옵션 🪄
공식 문서에 따르면, 이 컨테이너들에는 **`on_change`라는 파라미터(옵션)**를 추가해서 아주 똑똑하게 사용할 수 있다.

* **상태 추적 (Track state):** 사용자가 탭을 이동했는지, 확장 패널을 열거나 닫았는지 앱이 기억하고 추적할 수 있다.
* **앱 다시 실행 (Trigger reruns):** 탭을 누르거나 패널을 열고 닫을 때마다 앱 전체가 자동으로 다시 실행되면서 화면을 새롭게 업데이트해 준다.

---

## 3. 💻 코딩 예제
파이썬의 `with` 구문을 활용하면 이 컨테이너들을 아주 쉽게 화면에 그릴 수 있다. 

### ① 탭(`st.tabs`) 기본 사용법
여러 개의 화면 메뉴 창을 만들고 싶을 때 사용한다.

```python
import streamlit as st

# 2개의 탭 이름표 만들기
tab1, tab2 = st.tabs(["강아지 탭", "고양이 탭"])

# 첫 번째 탭에 내용 넣기
with tab1:
    st.write("여기는 귀여운 강아지 이야기를 적는 곳이야!")

# 두 번째 탭에 내용 넣기
with tab2:
    st.write("여기는 예쁜 고양이 이야기를 적는 곳이야!")
② 확장 패널(st.expander) 기본 사용법
클릭했을 때만 숨겨진 내용이 스르륵 나오게 하고 싶을 때 사용해.
import streamlit as st

# 클릭하면 열리는 상자 만들기
with st.expander("👉 여기를 클릭해서 비밀 메시지를 열어보세요!"):
    st.write("숨겨져 있던 마크다운 텍스트나 그림을 여기에 넣을 수 있어.")

```

# 💻 내 컴퓨터에 Streamlit 설치하기!

이 내용은 공식 문서의 **'시작하기(Get started) > 설치(Installation)'** 단계에서 안내하는 개발 환경 설정 과정이다.

## 1단계: 파이썬(Python) 준비하기
Streamlit은 파이썬 위에서 움직이는 도구이기 때문에, 컴퓨터에 파이썬이 먼저 설치되어 있어야 한다.

## 2단계: 까만 창(터미널/명령 프롬프트) 열기
컴퓨터에 명령을 내릴 수 있는 까만 창을 열어준다.
* **윈도우(Windows)**: 'cmd' 또는 '명령 프롬프트'를 검색해서 실행
* **맥(Mac)**: '터미널(Terminal)'을 검색해서 실행

## 3단계: 마법의 설치 주문 입력하기
창을 열었다면, 아래의 명령어를 똑같이 입력하고 키보드의 엔터(Enter)를 누른다.

```bash
pip install streamlit
```

## 4단계: 설치 확인
설치가 무사히 끝났다면, 아래 명령어를 입력해서 잘 작동하는지 테스트해 볼 수 있다.

```bash
streamlit hello
```
